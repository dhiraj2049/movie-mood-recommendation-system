# 🎬 VIBE — Find Movies by Mood

# 👨‍💻 About This Project

This project was created with love for cinema, design, and the feeling of finding the right movie at the right time.

Sometimes you don’t know what genre you want.

You just know the feeling.

Maybe you want something dark and intense.
Maybe something warm and comforting.
Maybe a strange late-night movie that stays in your head.

**VIBE** helps you discover movies based on mood, atmosphere, and feeling — not just categories.

> **What vibe are you in the mood for tonight?**

---

## ✨ What This Project Does

VIBE recommends movies based on emotional tone.

Instead of asking:

* Action or Comedy?
* Horror or Romance?

It helps you choose from moods like:

* 🔥 **Gritty & Raw**
* 🌑 **Noir & Dark**
* 🌻 **Feel-Good & Warm**
* ⚡ **Neon & Cyberpunk**
* 🎭 **Offbeat & Quirky**
* 🌀 **Surreal & Dreamlike**

---

## 🎥 Why I Built It

Most movie recommendation apps focus on genre, popularity, or ratings.

But many times, that’s not how people choose movies.

Sometimes you just want:

* a rainy-night film
* something emotional
* a stylish thriller
* a comfort watch
* a weird masterpiece

This project was built around that idea.

---

## 🛠️ How It Works

VIBE studies movie descriptions, keywords, and themes to understand the overall feeling of a film.

Then it recommends titles that match the vibe you choose.

It also lets you:

* Search for movies
* Find similar films
* Explore new moods
* Discover titles you may have missed

## 🚀 Run It Locally

### 1. Clone the project

```bash id="mk29zc"
git clone https://github.com/yourusername/vibe.git
cd vibe
```

### 2. Install requirements

```bash id="p3a8xr"
pip install -r requirements.txt
```

### 3. Add your TMDB API key and fetch movies

```bash id="f2n6kd"
python fetch_data.py --api-key YOUR_KEY
```

### 4. Start the app

```bash id="v4s1me"
uvicorn app:app --reload
```

### 5. Open in browser

```bash id="j7w2lu"
http://localhost:8000/app
```
